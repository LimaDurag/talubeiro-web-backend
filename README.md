## Getting Started

### 💻 Prerequisites

- [React](https://reactjs.org)
- [NodeJS](https://nodejs.dev/download)
- [Yarn](https://yarnpkg.com/getting-started/install)

### 🚀 Installation

1. Clone the repo
   ```sh
   git clone -b develop git@github.com:LimaDurag/talubeiro-web-react.git
   ```
2. Install Packages
   ```sh
   yarn install
   ```
